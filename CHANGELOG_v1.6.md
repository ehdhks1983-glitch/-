# GIF Maker Pro — v1.6.0 변경 사항

## 🔤 쇼츠 자막 — 전문가급(ASS/libass)으로 대공사

자막이 "그냥 글씨 박은" 수준이라 퀄리티가 낮던 문제를, 진짜 쇼츠 툴들이 쓰는 **ASS/libass 자막**으로 교체했습니다.

### 바뀐 점
- **크고 굵은 글씨 + 두꺼운 검정 외곽선 + 그림자** → 어떤 배경에서도 또렷하게.
- **등장 애니메이션** — 자막이 살짝 커지며 페이드로 "톡" 나타남(쇼츠 특유의 느낌).
- 자막 크기 슬라이더로 크기 조절(글씨가 1.7배 크게 매핑돼 시원하게 보임).
- 자막을 **사진에 굽지 않고 영상 위에 전문 렌더링으로 입힘** → 훨씬 선명.

### 작동 방식
- 합성 시: 사진/배경/워터마크로 영상을 만들고 → 마지막에 **ffmpeg ASS 자막을 영상 위에 번인**.
- libass가 없는 ffmpeg 환경이면 자동으로 기존(PIL) 방식으로 폴백 → 끊김 없음.
- 한글 폰트 자동 선택(윈도우 = 맑은 고딕 / 그 외 = 나눔고딕).

### 미리보기 참고
- 왼쪽 미리보기는 빠른 확인용이라 예전 스타일로 보일 수 있어요. **실제 출력(만들기 결과)이 새 ASS 자막**입니다.

## ✅ 검증
- libass 사용 가능 확인, ASS 파일(스타일+애니메이션 태그) 생성 확인
- **검정 영상에 자막 번인 → 자막 영역에 실제로 글씨가 그려지는지 픽셀로 확인**(밝은 픽셀 19,000+)
- 전체 쇼츠 빌드(ASS 경로) 1080×1920 정상, 전체 회귀 15종 통과
- ⚠️ 글꼴 모양·색감 등 최종 '눈으로 보는' 품질은 윈도우에서 한 번 확인 권장(저는 영상 직접 못 봐요).
