# 영상 제작기 (Shorts Maker)

GIF 메이커에서 **분리된 독립 프로그램**입니다. 유튜브 쇼츠/세로 영상 제작 전용.

## 구성
- **📝 영상 기획** — 목적·타겟·상품만 넣으면 후킹·대본·자막·장면표·검수까지 자동
- **📱 쇼츠 제작** — 기획 결과를 바로 받아 사진 + 자막(ASS) + 나래이션(음성) + 배경음악 → 세로 9:16 MP4
- 💧 워터마크 — 결과물에 블로그 ID/로고 자동 삽입

## 실행 방법
1. 이 폴더(영상 제작기)를 GIF 메이커와 **별도 폴더**로 둡니다.
2. **`ffmpeg` 폴더를 이 폴더 안에 복사**합니다. (GIF 메이커 폴더의 `ffmpeg`를 그대로 복사하면 됩니다 — 영상/음성 합성에 필요)
3. 실행: `python main.py`
   - 빌드해서 배포하려면 GIF 메이커의 `build.bat`를 참고해 `main.py` 기준으로 빌드하세요.

## 나래이션 음성 (선택, 권장)
- 윈도우 기본 음성은 로봇 같아서, **🎙 AI 음성 설정**에서 **ElevenLabs** API 키 + Voice ID를 넣으면 사람 같은 음성이 됩니다.
- 키가 없으면 윈도우 기본 음성으로 동작합니다.

## 필요 패키지
`pip install -r requirements.txt` (customtkinter, Pillow)

## 참고
- 설정(워터마크/음성)은 이 폴더의 `data/` 에 저장됩니다 (GIF 메이커와 독립).
- 라이선스(HWID) 잠금을 붙이려면 GIF 메이커의 `license_manager.py` / `license_dialog.py`를 복사해 `main.py`에 연결하면 됩니다. (기본은 잠금 없음)
