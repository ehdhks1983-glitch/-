# GIF Maker Pro — v1.5.0 변경 사항

쇼츠가 "만들어지긴 하는데 못 올릴 수준"이던 문제(로봇 음성 + 거친 배경음악)를 해결했습니다.

## 🎙 나래이션 — ElevenLabs 자연스러운 음성
- 윈도우 기본 음성(SAPI)은 로봇 같아 유튜브엔 부족 → **ElevenLabs 연동** 추가.
- 쇼츠 탭 나래이션 옆 **`🎙 AI 음성 설정`** 버튼 → API 키 + Voice ID 입력 → 사람 같은 한국어 음성.
- **키가 없으면 기존 기본 음성으로 자동 폴백**(앱은 그대로 동작). 설정은 `data/tts.json`에 로컬 저장(배포 안 됨).
- 설정창에서 **🔊 음성 테스트** 버튼으로 바로 들어보고 확인 가능.
- 안정성/유사도 슬라이더로 톤 조절.

> ElevenLabs 최저 플랜 월 $5~. 사이트에서 API 키와 Voice ID를 복사해 넣으면 됩니다.
> (도완님이 보내주신 영상에서 쓰던 바로 그 방식이에요.)

## 🎵 배경음악 믹스 개선
- **페이드 인/아웃** — 음악이 뚝 시작·끝나지 않고 자연스럽게.
- **자동 더킹(sidechain)** — 나래이션이 나올 때 배경음악이 자동으로 작아져 **목소리가 또렷**하게 들림.
- **리미터** — 소리가 깨지지(클리핑) 않게.
- 기본 배경음악 볼륨을 **25% → 18%**로 낮춤(음성 우선).

## ⚙️ 참고
- ElevenLabs 호출은 인터넷이 필요합니다. 실패하면 자동으로 기본 음성으로 대체돼 쇼츠는 계속 만들어집니다.
- 더킹/페이드 필터가 안 되는 환경이면 단순 믹스 → 나래이션만 순으로 자동 폴백(끊김 없음).

## ✅ 검증
- TTS 설정 저장/활성조건, 키 없을 때 안전 폴백, 시스템 음성 폴백 — 통과
- **새 배경음악 믹스(페이드+더킹+리미터)가 실제 ffmpeg에서 동작** + 전체 쇼츠 빌드(오디오 포함 1080×1920) — 통과
- 전체 회귀 14종 통과, 6탭 레이아웃 충돌 0건
- ⚠️ ElevenLabs **실제 음성 품질**은 키가 있어야 확인 가능 — 키 넣고 🔊 테스트로 들어봐 주세요.
