# GIF Maker Pro — v1.9.0 (✨ 사진 → 움짤)

사진 **1장**으로 영상처럼 움직이는 움짤을 만드는 새 탭을 추가했습니다.

## 새 탭: ✨ 사진 → 움짤
- 사진 1장 + 움직임 효과 → GIF 또는 WebP
- **효과**: 켄번스(확대+이동), 줌인, 줌아웃, 좌·우·상·하 이동(패닝)
- 길이(2~6초) · FPS(12~24) · 확대량(1.1~1.8x) · 화질 프리셋(최고화질/균형/빠른로딩) · 저장 위치
- 부드러운 가감속(ease), **워터마크 자동 적용**, GIF 글로벌 팔레트 + gifsicle 마무리 재사용
- **무료·오프라인** (ffmpeg 불필요, PIL만 사용)

## 참고 (솔직하게)
- "카메라가 움직이는" 효과입니다 — 사진 속 **인물 자체가 움직이진 않습니다**(그건 유료 클라우드 AI가 필요).
- **WebP 출력이 GIF보다 훨씬 작습니다**(로딩 빠름) → 블로그·웹은 WebP 추천.

## 검증
- 5개 효과 GIF + WebP 실제 렌더링 OK, 켄번스 첫↔끝 프레임이 실제로 달라짐(움직임 발생)
- 5탭 앱 구성 + grid/gui 회귀 통과
