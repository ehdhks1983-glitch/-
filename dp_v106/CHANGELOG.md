## v21.8.24.107 (manifest 1.6.1) — 오류 수정판

1. **[버그] 아이콘 클릭 동적 주입 누락 수정** (`background.js`)
   - ChatGPT 탭이 확장 설치/리로드보다 먼저 열려 있던 경우, 아이콘 클릭으로 주입되는 경로에서
     `gif_encoder.js / zip_store.js / template_store.js / fix_panel.js` 4개가 빠져
     움짤·ZIP 묶음 내보내기·템플릿·고치기 패널이 동작하지 않던 문제.
   - 주입 목록을 manifest `content_scripts`와 1:1 동일하게 맞춤.
2. **[품질] 카테고리 기본 카피 일반화** (`product_analyzer.js`, `prompt_short_dynamic.js`)
   - 의류 폴백/전략이 래쉬가드(워터파크·물놀이) 전용, 잡화 폴백/전략이 명함지갑 전용으로 굳어 있어
     다른 옷·가방·모자 등에 엉뚱한 문구가 주입되던 문제 → 카테고리 일반 전략으로 교체.
   - GIFT 섹션 역할 문구도 명함 전용 → 일반 선물 장면으로 교체.
   - 분류 규칙(래쉬가드→의류, 명함지갑→잡화 등)은 그대로 유지(회귀 없음, 테스트로 검증).
3. **[규격] manifest description 132자 제한 준수** — 변경 이력을 CHANGELOG.md로 분리.
4. **[버그] 상품명 이모지 깨짐 수정** (`link_parser.js`)
   - HTML 수치 문자참조 디코딩이 `String.fromCharCode` 기반이라 BMP 밖 문자(이모지 등)가
     깨지던 문제 → 안전한 코드포인트 디코딩으로 교체(비정상 코드포인트는 무시).
5. **[테스트] 가상 테스트 러너 추가** (`tests/run_tests.js`)
   - 크롬 없이 Node(vm)로 모듈을 로드/실행해 검증: 주입 일관성, 분류기, 프롬프트 엔진,
     manifest 규격, 링크 파서, GIF/ZIP 인코더. `node tests/run_tests.js`로 실행.
