# GIF Maker Pro — v1.4.0 변경 사항

## 📝 새 탭: 영상 기획 디렉터 (맨 앞 + 기본 탭)

"무슨 말을 해야 하지?"에서 막히지 않게, **입력 몇 개 → 완성된 영상 기획서**를 뽑아주는 모듈입니다.
대본 생성기가 아니라, 유튜브 쇼츠/롱폼의 **기획 순서 전체를 끌고 가는 디렉터**예요.

### 입력 (왼쪽)
① 영상 목적(조회수/신뢰/문의/판매/교육) · ② 타겟 · ③ 상품/서비스 · ④ 고객 문제(💡예시 버튼)
· ⑤ 핵심 메시지(비우면 자동) · ⑥ 흔한 오해 · ⑦ 후킹 유형 · 길이(쇼츠 15/30/60·롱폼)

### 출력 — '🎬 전체 기획서 생성' 한 번에 (오른쪽)
1. 영상 목적 / 2. 타겟 / 3. 상품 / 4. 고객 문제 / 5. **핵심 메시지(딱 하나)**
6. **후킹 후보 6종**(공감·경고·반전·질문·사례·숫자) / 7. **영상 구조**(시간 구간별)
8. **대본** / 9. **장면 구성표**(시간·화면·자막·소스) / 10. **자막**(한 줄씩 짧고 크게)
11. **업로드 패키지**(제목 5개·설명·해시태그·고정댓글) / 12. **검수 리포트(점수 + 개선점)**

→ **📋 복사** 로 메모장·블로그·영상편집기에 바로 붙여넣고, **💾 저장**으로 .txt 보관.

### 설계 포인트
- "AI 파는 채널"이 아니라 **"반복 업무를 줄여주는 법을 알려주는 채널"** 포지션에 맞춰
  **판매형이 아니라 교육형**(문제→공감→원인→해결→자연스러운 CTA) 구조로 뽑습니다.
- 검수기가 첫1초 후킹·한 메시지·광고냄새·CTA 자연스러움 등을 점수화하고 개선안을 제시합니다.
- **한글 조사(이/가·로/으로)를 받침에 맞춰 자동 보정** — 어색한 문장이 안 나오게.

### 중요
- **오프라인 동작** — 별도 설치·인터넷·API 키·비용이 필요 없습니다(배포 시 구매자도 그대로 사용).
  더 똑똑한 AI 생성을 원하면 나중에 API 연동 옵션을 붙일 수 있게 엔진(`content_planner.py`)을 분리해 뒀습니다.
- 기존 5개 탭/기능은 **전혀 건드리지 않았습니다**. '📝 영상 기획'은 새 탭으로만 추가.
