  - '💬 자막' 섹션이 '📦 GIF 용량 줄이기' 섹션과 **같은 위치(grid 15·16행)에 겹쳐** 표시되던 문제를 수정.
  - 자막 섹션과 그 아래 위젯(출력 폴더·변환 버튼·진행률·상태)을 17~26행으로 재배치.

## ✨ 새 기능

### 1. 🟡 카카오톡 이모티콘 원클릭 (편집 탭)
- 버튼 한 번으로 기존 GIF/WebP/APNG를 **카톡 이모티콘 규격**으로 변환.
  - 정사각형(1:1) 크롭 → **360×360** → **3초 이내**로 자동 컷 → **WebP** → **3MB 이하** 자동 압축.
- 엔진: `editor.make_kakao_emoticon()`, 길이 제한 헬퍼 `editor.trim_to_duration()`.

### 2. 📱 MP4 · 쇼츠(세로 9:16) 출력 (영상 탭)
- 출력 포맷에 **MP4(H.264 + AAC)** 추가 — **원본 오디오 유지**, 속도 변경 시 오디오도 같이 조절(atempo).
- **쇼츠 체크박스**: 세로 **1080×1920(9:16)** 캔버스에 **블러 배경 + 가운데 맞춤**으로 합성 → 인스타·유튜브 쇼츠/릴스용.
- 자막·구간 자르기·자동 분할과 모두 호환. (쇼츠 옵션은 MP4 선택 시에만 활성화)
- 엔진: `video_converter._convert_to_mp4()`, `ConvertJob.shorts_vertical`.

### 3. 🗂 배치(여러 파일) 변환 (편집 탭)
- '📂 여러 개'로 파일을 여러 개 선택 → '🗂 일괄 변환' 한 번으로 **현재 편집 설정 + 출력 포맷을 전부에 적용**.
- 파일 하나씩 순차 처리(메모리 안전), n/총개수 진행률 표시.

### 4. ▶ 결과 미리보기 (편집 탭)
- 변환이 끝나면 **'▶ 결과 미리보기'** 버튼이 떠서, 만들어진 결과물을 앱 안에서 바로 불러와 재생.
