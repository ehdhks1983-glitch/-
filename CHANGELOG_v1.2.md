# GIF Maker Pro — v1.2.0 변경 사항

## 🎬 새 탭: "쇼츠 제작" (세로 9:16 영상 메이커)

사진 여러 장으로 **유튜브 쇼츠 · 인스타 릴스용 세로 영상**을 만드는 전용 탭이 추가됐습니다.

### 들어간 것
- **장면 기반 구성**: 사진을 추가하면 한 장이 한 '장면'이 됩니다. 순서 이동(🔼🔽)·삭제 가능.
- **🗣 나래이션 (글 → 음성)**: 장면마다 읽어줄 글을 입력하면 **음성으로 변환**해 깔아줍니다.
  - 윈도우 **내장 음성(한국어)** 사용 — 별도 설치·인터넷·비용 **불필요**.
  - 나래이션이 길면 그 장면 노출 시간을 **자동으로 늘려** 말이 잘리지 않습니다.
- **🎵 배경음악(BGM)**: 본인이 가진 음악 파일을 골라 깔 수 있고, **볼륨 조절** 가능. (나래이션과 자동 믹스)
- **💬 화면 자막**: 장면마다 자막 입력(자동 줄바꿈·외곽선·반투명 배경). 크기 조절.
- **🎨 구조(템플릿) 3종**:
  - `흐림 배경` — 사진을 가운데 두고 배경은 흐리게 (가장 무난)
  - `꽉 채움` — 사진을 9:16로 꽉 차게 크롭
  - `카드뉴스` — 흰 배경 + 상단 큰 제목 + 사진
- **미리보기**: 선택한 장면이 실제 9:16 화면으로 어떻게 나오는지 바로 확인.
- **출력**: 1080×1920 **MP4** (H.264 + AAC, 나래이션·배경음악 합성 완료).

### 사용 순서
1. `📱 쇼츠 제작` 탭 → `➕ 사진 추가`
2. 장면을 클릭 → 우측에서 **구조 / 자막 / 나래이션 / 노출시간** 설정 → `🔄 적용 + 미리보기`
3. (선택) `🎵 음악 선택`으로 배경음악 + 볼륨
4. `🚀 쇼츠 만들기` → 출력 폴더에 세로 MP4 생성

### 곁다리 개선 (영상 탭)
- `📱 쇼츠로 만들기` **원클릭 버튼** 추가 — 기존 영상 1개를 세로 쇼츠로 바로 변환(포맷+쇼츠 한 번에 설정).

## 📝 참고
