# GIF Maker Pro — v1.3.0 변경 사항

전체 코드 점검(정적 분석 포함) 후, **꼭 필요한 버그 수정**과 블로그에 핵심인 **워터마크 기능**을 넣었습니다.

---

## 💧 새 기능: 워터마크 (블로그 ID·로고 자동 삽입)

설명서엔 안내돼 있었지만 **실제로는 빠져 있던 기능**을 구현했습니다. 블로그 이미지·움짤 브랜딩에 핵심이에요.

- 상단 **`💧 워터마크`** 버튼 → 설정 창 (실시간 미리보기 포함)
- **텍스트**(@블로그아이디) / **로고 이미지(PNG)** / **둘 다** 선택
- 위치(4모서리·가운데), 크기, 투명도, 여백, 글자색 조절
- **모든 변환 결과에 자동 적용**:
  - 이미지 합치기 · 편집 · 쇼츠 제작 → 텍스트 + 로고 이미지
  - 영상 → 움짤 → 텍스트 워터마크 (drawtext)
- 한 번 설정하면 계속 유지(`data/watermark.json`에 저장).

> 사용법: `💧 워터마크` → '워터마크 사용' 켜기 → 종류/내용/위치 설정 → 미리보기 확인 → '저장하고 닫기'. 이후 모든 변환에 자동으로 찍힙니다.

---

## 🐛 버그 수정 (정적 분석으로 발견한 실제 버그)

1. **편집 탭 텍스트 한글 깨짐** — `편집` 탭의 '텍스트' 기능이 영문 전용 폰트(arial)를 써서 **한글이 깨지던** 문제를, 한글 폰트(맑은 고딕/나눔) 자동 적용으로 수정.
2. **편집 실패 시 오류 메시지가 안 뜨던 버그** — 편집/저장이 실패하면 에러 안내 대신 내부적으로 `NameError`가 나던 문제(예외 변수 처리 오류)를 수정.
3. **배치 변환 포맷 함정** — '여러 개'로 파일을 고르면 출력 포맷이 **첫 파일 포맷으로 바뀌던** 불편함을, 사용자가 고른 포맷을 유지하도록 수정. (GIF 여러 개 → WebP 일괄 변환이 자연스러워짐)

> 그 외 사용에 큰 영향 없는 항목(미사용 import 등)은 의도적으로 손대지 않았습니다.
