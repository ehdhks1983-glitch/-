# 추가 진행 리스트 (사용자 요청 — 잊지 말 것)

기록일: 2026-06-10 (v105 시점)

## 1. 제품 충실도 (연출샷 오류)
- 증상: 제품 이미지를 기반으로 한 연출샷에서 **간혹 제품과 다른 형태/색의 제품**이 그려짐.
- 방향: 이미지 프롬프트의 제품 재현 기준 강화(원본 대조 강제·형태/색/로고 불일치 시 실패 명시),
  생성 후 자동 QA에서 제품 불일치 감지 시 재생성.

## 2. 자동제작 전 "기획안 선택" 단계 신설 (가장 중요)
- 요청: 자동제작 버튼을 누르기 **전에 한 단계** —
  제품에 맞는 **상황별 기획안 5종**을 먼저 제안 → 사용자가 1개 선택 → 그 기획안으로 자동제작.
- 예: 명함지갑이면 ① 첫 미팅 신뢰 ② 선물용 ③ 가죽 질감/장인 ④ 매일 휴대(수납·슬림) ⑤ 비교(일반 지갑 vs) 같은
  서로 다른 컨셉 5종.
- UI: 자동제작 흐름에 "기획안 5종 받기 → 선택 → 진행" 단계 삽입(건너뛰기 옵션 포함).

## 3. 연출 다양성 (매번 같은 연출)
- 증상: 명함지갑 = "미팅 가서 건넨다" 식으로 **매번 같은 상황 연출**만 나옴.
- 방향: 카테고리별 상황 연출 풀을 여러 개 두고 시드 회전(이미 무드는 회전 중 — 상황/장면도 회전),
  2번(기획안 5종)과 연계하면 자연 해결되는 부분 큼.

- 1차 작동 수리 완료(v105: 자가진단·실패 사유 표시·자동 수집).

- 3차: 사용자 실물 테스트 후 유지/제거 결정 (제거는 5분 작업).
